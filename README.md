# Workshop de Actions

O workshop de 2 horas sobre uso avançado de GitHub Actions, incluirá uma introdução aos conceitos básicos, mas foca em tópicos mais avançados:

### Título do Workshop: Workshop Avançado de GitHub Actions

#### Duração do Workshop: ~2 Horas

### Agenda:

#### Introdução (5 minutos)
- **Boas-vindas e Objetivos**
  - Breve introdução ao GitHub Actions (slides 1 a 4).
  - Descrever os objetivos do workshop.
  - Explicar a agenda.

#### Visão Geral Básica do GitHub Actions (15 minutos)
- **Introdução ao GitHub Actions**
  - Componentes básicos: workflows, jobs, steps e actions (slides 5 a 12).
- **Criando Sua Primeira Action**
  - Demonstrar um pipeline simples de CI/CD.
  - Conceitos-chave: diretório `.github/workflows`, sintaxe YAML 

#### GitHub Actions Avançado (90 minutos)
- **Workflows Complexos**
  - Jobs paralelos e sequenciais.
  - Builds em matriz.
  - Workflows reutilizáveis.

- **Actions Personalizadas**
  - Escrevendo actions personalizadas em JavaScript/TypeScript.
  - Escrevendo Actions em contêiner
  - Escrevendo Actions compostas
  - Publicando actions privadas para reutilização.

- **Segurança e Melhores Práticas**
  - Gerenciamento de segredos e variáveis.
  - Endurecimento de segurança para workflows (ex: definir permissões para editar workflows, definir escopos para runners auto-hospedados, grupos de runners)
  - Workflows reutilizáveis vs Actions Compostas

- **Configuração Avançada**
  - Variáveis de ambiente.
  - Execução condicional.
  
#### **Perguntas e Respostas e Encerramento (10 minutos)**
- Abrir espaço para perguntas.
- Recapitular pontos-chave (usar [apresentação de slides](slides/Actions_Workshop.pptx)).
- Fornecer recursos adicionais para aprendizado ([GitHub Automation Pathways](https://resources.github.com/learn/pathways/automation/)).
